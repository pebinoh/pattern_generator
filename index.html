Générateur de pattern

Cet outil vous permet de créer un patron de couture simple. Visualisez le résultat en 3D et générez le patron 2D correspondant.

Comment ça marche ?

Ajustez les dimensions dans le panneau de droite. L'aperçu 3D se met à jour en temps réel. Cliquez sur "Générer le patron !" pour voir le plan de découpe.
